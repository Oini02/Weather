# Weather App

## Setup

1. Clone repo  
2. `cd backend` then `npm install`  
3. Create `.env` with your OpenWeatherMap API key:  
   ```
   OPENWEATHER_KEY=YOUR_KEY_HERE
   ```
4. Run server:  
   ```
   node server.js
   ```
5. Visit http://localhost:3000

## Deployment

- Host backend on Heroku/Vercel
- Serve static frontend similarly
